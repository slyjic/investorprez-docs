üöÄ Quick Start

Installing Plugins

npm i @grapesjs/studio-sdk-plugins

Basic Usage

import { pluginName } from '@grapesjs/studio-sdk-plugins';

const config = {
  plugins: [
    pluginName,
    // or with options
    pluginName.init({ /* options */ })
  ]
};

üì¶ Plugin Categories

1. Layout Plugins

Manage component positioning and layout structures - Absolute Mode - Fixed positioning - Grid Mode - CSS Grid layouts

2. Component Plugins

Add new component types to the editor - Iconify - Icon components - Table - Table structures - Animation - Animated elements

3. Enhancement Plugins

Extend editor functionality - Sidebar Buttons - UI extensions - Rich Text Editor - Content editing

4. Integration Plugins

Connect with external libraries - Custom Renderer - React integration - Asset providers - Data sources
